int affichage ( Partie *  partie)

Lance l'affichage graphique SDL avec contrôle du robot au clavier (z,q,s,d).

Affiche l'entrepôt avec SDL2.

Initialise SDL, crée la fenêtre et le renderer, puis boucle sur les événements clavier pour déplacer le robot. Affiche en continu l'entrepôt.

Paramètres
partiePointeur vers la structure Partie contenant l'état de l'entrepôt.
Renvoie
int 0 si succès, 1 en cas d'erreur SDL.
